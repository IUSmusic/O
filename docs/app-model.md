# App Model

## Mother O apps

Mother O runs:
- core system apps
- bootstrap-generated apps
- optional ordinary apps
- user-generated apps

## Rules

- core system apps are not removable
- bootstrap-generated apps begin from template placeholders or first-run flows
- optional and user-generated apps are removable
- app creation is visible and traceable through terminal and ledger events

## Template-first behavior

O favors template-driven creation over shipping bloatware. A user should be able to generate useful apps quickly from The Library.
