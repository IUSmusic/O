# Install and Recovery

## Install model

O should support:
- developer-oriented flashing first
- future controlled normal-user installation path

## Recovery model

Recovery must allow:
- safe boot
- rollback
- reinstall
- preserve or wipe choices
- trusted restoration path

## Reinstall choices

The user should be able to choose:
- full wipe
- preserve user data only
- preserve selected Sub O only
- preserve templates only
- selective preservation

## Activation model

Important system changes must use:
- staged activation
- validation
- rollback-safe boot switching
