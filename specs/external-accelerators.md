# External Accelerators

## Goal

O should support optional external devices that improve inference, media, or desktop-style workflows without requiring the user to open the phone.

## Device classes

- accelerator.ai
- accelerator.audio
- accelerator.midi
- accelerator.display
- accelerator.compute

## Initial focus

1. USB audio
2. USB MIDI
3. USB AI accelerators
4. display/dock support
5. more advanced external compute later

## Rules

- devices are discoverable
- access is permission-governed
- exposure to Sub O occurs through the Device Gateway
- use is visible in the privacy ledger
