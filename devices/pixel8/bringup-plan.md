# Pixel 8 Bring-up Plan

## Phase 1
- identify exact supported retail model
- verify unlockability
- document flashing path
- confirm partition layout
- establish signed image workflow

## Phase 2
- boot custom kernel + initramfs
- mount O root filesystem
- validate display, touch, storage

## Phase 3
- validate Wi-Fi, USB host, charging, audio
- establish signed updates and rollback
- boot Mother O shell

## Phase 4
- add Sub O lifecycle
- add privacy ledger
- add local model runtime
- add USB audio / MIDI support

## Phase 5
- evaluate telephony and cellular data support
- harden trust and update path
