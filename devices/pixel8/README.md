# Pixel 8 Reference Target

## Reference device

Google Pixel 8 (unlocked retail model)

## Objective

Bring O to one real modern phone as a proof of concept and demonstrate the essential O experience on that device.

## Proof gate

A successful first proof requires:
- boot and recovery
- display and touch
- storage
- Mother O shell
- terminal and file manager
- The Library offline
- browser/bootstrap creation flow
- Sub O lifecycle
- local AI and brokered online AI
- Wi-Fi
- USB host
- audio
- rollback/recovery path
